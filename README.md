# List Editor

![Test and Deploy](https://github.com/Madmaxim22/List-Editor/workflows/Test%20and%20Deploy/badge.svg)

List Editor - это веб-приложение для управления списком товаров с возможностью добавления, редактирования и удаления записей. Приложение реализовано с использованием чистого JavaScript, HTML и CSS с применением архитектурного паттерна MVC (Model-View-Controller).

## Функциональность

- Добавление новых товаров с указанием названия, описания и цены
- Редактирование существующих товаров
- Удаление товаров с подтверждением действия
- Валидация вводимых данных (проверка на заполнение названия и корректность цены)
- Интуитивно понятный пользовательский интерфейс с модальными окнами
- Адаптивный дизайн для комфортной работы на разных устройствах

## Технологии

- JavaScript (ES6+ модули)
- HTML5
- CSS3
- Webpack (сборка проекта)
- Jest (модульное тестирование)
- Puppeteer (интеграционное тестирование)

## Архитектура

Проект структурирован по архитектурному паттерну MVC:
- **Model** (`ProductModel.js`) - управление данными товаров
- **View** (`View.js`) - отображение данных и пользовательский интерфейс
- **Controller** (`ProductManager.js`) - логика взаимодействия между Model и View

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <ссылка_на_репозиторий>
cd list-editor
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение в режиме разработки:
```bash
npm start
```

Приложение будет доступно по адресу http://localhost:5000

## Сборка проекта

Для создания production-сборки выполните:
```bash
npm run build:prod
```

Для разработки:
```bash
npm run build:dev
```

## Тестирование

Запуск модульных тестов:
```bash
npm run test:unit
```

Запуск интеграционных тестов:
```bash
npm run test:integration
```

Покрытие кода тестами:
```bash
npm run test:unit:coverage
```
